Magellan topbar

Responsive topbar with magellan and sticky behavior.

Magellan component is only about to track scrolling and targets (the element to trigger active classname on menu entries). Like in this demo, you need to combine menu, sticky and magellan components to achieve the same look and feel from Foundation5.
